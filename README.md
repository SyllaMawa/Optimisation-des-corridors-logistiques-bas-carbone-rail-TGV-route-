# Optimisation des corridors logistiques bas carbone rail TGV route
Amazon cherche à réduire l’usage du camion sur les trajets intersites tout en gardant des délais de livraison très courts. Avec l’arrivée du corridor TGV Paris–Lyon et l’augmentation du transport ferroviaire, il devient essentiel d’identifier où déployer les prochains axes bas carbone.
Ce projet utilise uniquement des données ouvertes L'analyser les flux potentiels et proposer les corridors les plus pertinents.


#🎯 Objectif
Identifier les trajets où Amazon pourrait remplacer le transport routier par du rail ou du TGV, afin de réduire les émissions de CO₂ tout en maintenant la rapidité des livraisons.

#🧩 Données utilisées
OpenStreetMap : réseau routier, distances, vitesses.

SNCF Open Data : lignes ferroviaires, horaires, capacités.

ADEME / EEA : facteurs d’émissions par mode de transport.

Copernicus : densité urbaine, zones à faibles émissions.

#🔍 Ce que j’analyse
Les trajets intersites où les camions sont encore majoritaires.

Les corridors ferroviaires existants ou sous‑utilisés.

Le gain potentiel en CO₂ si un flux passe du camion au rail/TGV.

L’impact sur les délais de livraison.

#📊 Résultats attendus
Une carte des corridors prioritaires pour les prochaines années.

Un modèle de routage multimodal (route + rail + TGV).

Une estimation des émissions évitées.

Des recommandations opérationnelles réalistes pour Amazon.
